Welcome to Analyze My Mood!!!!

The purpose of this project is enabling users to enter a query and analyze the sentiment associated with that query. Sentiment analysis is a popular topic in natural language processing and has a wide range of application areas. Social media is one of the reasons of the popularity of sentiment analysis. Thanks to applications like Twitter, millions of users share their opinions about different topics almost every second. There is a rich information pool to be analyzed.

For example, for a marketing expert who has released a new product to the market, the tweets about their product can be considered much more important than the sales numbers. By analyzing the tweets, the reason of unsatisfactory of the customers can be detected and immediate actions can be taken. Sentiment analysis is also important for politicians. They can analyze the reaction of the voters about their actions and update their campaigns accordingly.

Twitter API

In this project Twitter is used as data source. Twitter API provides many opportunities to the users like analyzing past conversations, measuring tweet performance, listening for important events, exploring tweets for a specific account, etc.
